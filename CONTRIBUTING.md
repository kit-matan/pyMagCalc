# Contributing to pyMagCalc

Thank you for your interest in contributing to pyMagCalc!

## Reporting Bugs
If you encounter any issues, please open an Issue on GitHub. Include:
- Steps to reproduce the bug
- Expected behavior vs. actual behavior
- Your configuration file (if applicable)

## Feature Requests
We welcome ideas for new features! Please open a Discussion or Issue to describe your idea before starting implementation.

## Pull Requests
1. Fork the repository.
2. Create a new branch for your feature (`git checkout -b feature/amazing-feature`).
3. Commit your changes.
4. Push to the branch.
5. Open a Pull Request.

## Code Style
- Please follow PEP 8 for Python code.
- Run `pytest` to ensure no regressions were introduced.
